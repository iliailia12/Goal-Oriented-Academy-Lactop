# Level 96

### დავალება 1

მოცემულია ობიექტი:

```js
const scores = {
  math: 80,
  english: 75,
  science: 90
};
```

დაწერე ფუნქცია, რომელიც for...in ციკლით დაითვლის ყველა ქულის ჯამს.

---
                       









                       
### დავალება 2

მოცემულია:

```js
const user = {
  name: "Gio",
  age: 25,
  city: "Tbilisi",
  salary: 1200
};
```

for...in გამოყენებით დაბეჭდე მხოლოდ ის properties, რომელთა მნიშვნელობა არის რიცხვი.

---

### დავალება 3

შექმენი ფუნქცია, რომელიც იღებს ობიექტს და for...in გამოყენებით დააბრუნებს ყველა key-ს მასივის სახით.

---

### დავალება 4

მოცემულია:

```js
const student = {
  name: "Ana",
  age: 20,
  faculty: "Computer Science"
};
```

გამოიყენე დესტრუქცია ისე, რომ შექმნა ცვლადები name და faculty.

---

### დავალება 5

იგივე student ობიექტიდან ამოიღე name და age,
მაგრამ შეინახე ისინი ცვლადებში studentName და studentAge.

---

### დავალება 6

შექმენი ფუნქცია, რომელიც იღებს ასეთ ობიექტს:

```js
{
  width: 100,
  height: 50
}
```

დესტრუქციის გამოყენებით დააბრუნე ფართობი (width * height).

---

### დავალება 7

მოცემულია:

```js
const product = {
  title: "Laptop",
  price: 1500
};
```

დესტრუქციით ამოიღე title, price და discount,
სადაც discount default იყოს 0.

---

### დავალება 8

მოცემულია:

```js
const name = "Nino";
const age = 22;
```

შექმენი ობიექტი person ისე, რომ გამოიყენო Property Value Shorthand.

---

### დავალება 9

მოცემულია:

```js
const salaries = {
  Giorgi: 2000,
  Ana: 1800,
  Luka: 2200
};
```

for...in გამოყენებით დაბეჭდე:

Giorgi earns 2000
Ana earns 1800
Luka earns 2200

---

### დავალება 10

მოცემულია:

```js
const users = [
  { name: "Gio", age: 17 },
  { name: "Ana", age: 21 },
  { name: "Luka", age: 19 }
];
```

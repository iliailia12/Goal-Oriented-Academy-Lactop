1) კომენტარების სახით ახსენით რა განსხვავენბაა id selector-სა და class selector-ს შორის <3


გამოიყენება მხოლოდ ერთი უნიკალური ელემენტისთვის
გვერდზე ერთი და იგივე ID არ უნდა განმეორდეს
ყველაზე ძლიერი specificity CSS-ში
ხშირად გამოიყენება JS-ში კონკრეტული ელემენტის შესაცვლელად
იწყება #: #my-id

გამოიყენება ბევრ ელემენტზე ერთდროულად
class შეიძლება ბევრჯერ განმეორდეს გვერდზე
გამოიყენება ჯგუფური სტილებისთვის
იწყება წერტილით: .my-class

ID → უნიკალური, ერთი ელემენტი
Class → მრავალჯერადი, ჯგუფური ელემენტები

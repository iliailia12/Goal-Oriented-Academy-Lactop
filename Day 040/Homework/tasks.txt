1) კომენტარების სახით ახსენით რა განსხვავენბაა id selector-სა და class selector-ს შორის <3

2) შექმენით 5 ცალი button, აქედან css-ში ჩვეულებირვ წამოიღეთ tag selector-ით ყველა button და გასტილეთ ისინი (color; background-color). პირველ 3 button-ს შეუქმენით და გადაეცით ერთნაირი class, და class selector-ის მეშვეობით გასტილეთ ისინი სხვანაირად. 4-ე button-ს კი გადაეცით id, და გასტილეთ იგი უნიკალურად id selector-ის მეშვეობით, 5-ე button-ი კი დატოვეთ ისე :>

3) ივარჯიშეთ id selector-ების და class selectore-ის გამოყენებაზე <3

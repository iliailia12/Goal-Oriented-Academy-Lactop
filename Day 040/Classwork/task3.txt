3) კომენტარების სახით ახსენით რა განსხვავებაა class selector-სა და id selector-ს შორის.



CLASS (.class)
ბევრ ელემენტზე გამოიყენება
შეიძლება განმეორდეს
იწყება წერტილით

ID (#id)
მხოლოდ ერთ ელემენტზე

იწყება ღარლით #

განსხვავება:
class → ჯგუფისთვის
id → უნიკალური ელემენტისთვის

2) კომენტარების სახით ახსენით თქვენი სიტყვებით რა არის id selector და რა დანიშნულება გააჩნია, რისთვის ვიყენებთ?


ID selector არის CSS-ის სელექტორი,
რომელიც გამოიყენება იმისთვის,
რომ კონკრეტულ, ერთ-ერთ ელემენტს მივანიჭოთ სტილი.
ID selector უნიკალურია —
ერთ HTML გვერდზე ერთი და იგივე ID არ უნდა განმეორდეს.
ID selector-ს წინ უძღვება სიმბოლო "#"

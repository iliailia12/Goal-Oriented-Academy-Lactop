2) კომენტარებით ახსენით რა არის Boolean და binary code



 Boolean (ბულის ტიპი):
 Boolean არის მონაცემთა ტიპი, რომელიც შეიძლება მიიღებდეს მხოლოდ ორ მნიშვნელობას:
 True (სიმართლე) და False (სიყალბე).
 ეს ტიპი გამოიყენება პირობებში, შედარებებში და ლოგიკურ ოპერაციებში.




 Binary code (ბინარული კოდი):
 ბინარული კოდი არის მონაცემების წარმოდგენა მხოლოდ ორი სიმბოლოთი:0 და 1.
 კომპიუტერები შიგნით ყველაფერს — ტექსტს, სურათებს, მუსიკას, პროგრამებს —
 საბოლოოდ ბინარულ კოდად ინახავენ და ამუშავებენ.
 მაგალითად, ასო "A" ბინარულად შეიძლება იყოს: 01000001

1) კომენტარების სახით ჩამოწერეთ ყველა პოზიციონირების მნიშვნელობა და თითოეულის განმარტება.

// static — ელემენტის ნაგულისხმევი პოზიციონირება, არ ექვემდებარება top/right/bottom/left მახასიათებლებს.

// relative — ელემენტი პოზიციონირდება თავისი საწყისი პოზიციიდან შედარებით; top/right/bottom/left მისი საწყისი პოზიციის მიმართ გადაიწევს.

// absolute — ელემენტი პოზიციონირდება უახლოესი პოზიციონირებული (relative/absolute/fixed/sticky) მშობლის მიმართ; იგი გამოდის დოკუმენტის ჩვეული ნაკადიდან.

// fixed — ელემენტი ფიქსირდება viewport-ზე;滚ვისას არ მოძრაობს და ყოველთვის იმავე ადგილზე რჩება.

// sticky — ელემენტი behaves როგორც relative სანამ scrolling–ით მიაღწევს განსაზღვრულ პოზიციას, შემდეგ იქ “ეკრობა” და მუშაობს როგორც fixed.

### 🟦 `.forEach()` – 5 დავალება

2. გაქვს მასივი რიცხვებისგან:

   ```js
   const numbers = [1, 2, 3, 4, 5];
   ```

   გამოიყენე `.forEach()`, რომ კონსოლში დაბეჭდო თითოეული რიცხვი.

3. მოცემულია სახელების მასივი:

   ```js
   const names = ["John", "Sarah", "Mike"];
   ```

   `.forEach()`-ით დაბეჭდე ტექსტი:
   `Hello, John`, `Hello, Sarah` და ა.შ.

4. გაქვს რიცხვების მასივი. `.forEach()`-ის გამოყენებით დაითვალე და კონსოლში დაბეჭდე მათი ჯამი.

5. მოცემულია სიტყვების მასივი:

   ```js
   const words = ["javascript", "is", "awesome"];
   ```

   `.forEach()`-ით დაბეჭდე თითოეული სიტყვა და მისი სიგრძე.

6. გაქვს სტუდენტების სახელების მასივი. `.forEach()`-ით დაბეჭდე თითოეული სახელი ინდექსთან ერთად
   (მაგ: `0 - John`).

### 🟩 `.map()` – 5 დავალება

7. გაქვს რიცხვების მასივი. გამოიყენე `.map()`, რომ შექმნა ახალი მასივი, სადაც ყველა რიცხვი გამრავლებულია 2-ზე.

8. მოცემულია ტემპერატურების მასივი ცელსიუსში. `.map()`-ით გადაიყვანე ისინი ფარენჰეიტში
   ფორმულით: `F = C * 9/5 + 32`.

9. გაქვს სახელების მასივი. `.map()`-ის გამოყენებით შექმენი ახალი მასივი, სადაც ყველა სახელი დიდი ასოებით იქნება (`toUpperCase()`).

10. მოცემულია ობიექტების მასივი:

   ```js
   const users = [
     { name: "John", age: 20 },
     { name: "Sarah", age: 25 }
   ];
   ```

   `.map()`-ით შექმენი ახალი მასივი მხოლოდ სახელებით.

11. გაქვს რიცხვების მასივი. `.map()`-ის გამოყენებით შექმენი ახალი მასივი, სადაც თითოეული ელემენტი იქნება ტექსტი:
    `"Number: X"` (მაგ: `"Number: 5"`).

# 5) მოიძიეთ თუ რა განსხვავებაა print(5 + 5)-სა და print("5" + "5")-ს შორის და ახსენით კომენტარებით


# პირველი მაგალითი
print(5 + 5)  
# აქ ორივე მნიშვნელობა არის integer (რიცხვი).
# ოპერატორი + მათ შორის ნიშნავს შეკრებას.
# შედეგი იქნება 10.

# მეორე მაგალითი
print("5" + "5")  
# აქ ორივე მნიშვნელობა არის string (ტექსტი).
# ოპერატორი + სტრინგებზე ნიშნავს კონკატენაციას (შეერთებას).
# ამიტომ "5" + "5" გახდება "55".
